## Prérequis

Assurez-vous d'avoir sur votre ordinateur avant de suivre les étapes ci-dessous.
Angular CLI: 17.3.8
Node: 20.9.0
Package Manager: npm 10.8.1

## Installation

Git clone le projet avec la commande :

```bash
 Git clone
```

Installation des dépendances : 

```bash
npm i
```


Démmarage du server

```bash
npm start
```
